# Task 3: Heart Disease Prediction

## Overview
This project is part of the AI/ML Engineering Internship tasks at DevelopersHub Corporation. The objective of this project is to predict whether a patient is at risk of heart disease using medical and health-related attributes.

## Objective
The main goal of this project is to:
- Analyze medical patient data
- Perform Exploratory Data Analysis (EDA)
- Train a Machine Learning classification model
- Predict heart disease risk
- Evaluate model performance using classification metrics

## Dataset
- Dataset: Heart Disease Dataset
- Source: Kaggle / UCI Machine Learning Repository

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn

## Machine Learning Model
- Logistic Regression

## Project Workflow
1. Load and inspect dataset
2. Handle missing values
3. Perform Exploratory Data Analysis (EDA)
4. Visualize feature relationships
5. Select features and target variable
6. Split dataset into training and testing sets
7. Train Logistic Regression model
8. Predict heart disease outcomes
9. Evaluate model performance

## Evaluation Metrics
The model was evaluated using:
- Accuracy Score
- Confusion Matrix
- Classification Report
- ROC-AUC Score
- ROC Curve

## Key Insights
- Certain medical attributes strongly influence heart disease prediction.
- Logistic Regression performed effectively for binary classification.
- Data visualization helped identify important trends and feature relationships.

## Conclusion
This project demonstrates how Machine Learning can assist in medical prediction systems by analyzing patient health data and identifying heart disease risk factors.
